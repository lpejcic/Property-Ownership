# Udacity Blockchain Capstone
## Truffle version used in the project.

* Truffle v5.3.2 (core: 5.3.2)
* Solidity - 0.5.5 (solc-js)
* Node v14.15.5
* Web3.js v1.3.5
## To get started locally:

1. Clone this repo
1. Change into the project directory `cd Property-Ownership/`
1. Install dependenceis `npm i`
1. Start your local blockchain `ganache-cli`
1. Compile the contracts `truffle compile`
1. Migrate the contracts `truffle migrate --reset`
1. Run the tests `truffle test`
1. Change into eth-contracts directory `cd eth-contracts/`

## ZoKrates

This project uses ZoKrates for implementing zkSNARKS. To run ZoKrates in a Docker Container, run the following command:

```
docker run -v $pwd/code:/home/zokrates/code -ti zokrates/zokrates /bin/bash
```

In the running container terminal execute the following commands to compile the proof code, setup and export the verifier contract.

```
~/zokrates compile -i square.code
```

```
~/zokrates setup
```

```
~/zokrates export-verifier
```

Now generate unique proof ready for the NFT minting process:

```
~/zokrates compute-witness -a 1 1
```
```
~/zokrates generate-proof
```
